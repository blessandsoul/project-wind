A — Vertical Timeline

Cons

B — Facetune Style: Image-Dominant with Floating UI

Huge images fill each step. Step labels and UI controls float ON the images as glassmorphism overlays, like app screenshots. Steps connected by subtle arrows. Image is the star — text is minimal.

Pros

  • Most cinematic/premium feel
  • Images are the hero — matches Facetune
  • Floating UI overlays feel like real product
  • Full-width on mobile — native app feel

Cons

  • Step descriptions less prominent
  • More complex implementation

C — Tab-Switcher with Hero Image

Horizontal step pills at top (1→2→3). One large image area below that transitions between steps. Active step highlighted. Text description below the image. Like a product tour.

Pros

  • Very compact — no scrolling needed
  • Interactive feel (click tabs)
  • Single large image area

Cons

  • Requires client-side state/interactivity
  • Only shows one step at a time
  • Less scroll-based discovery
